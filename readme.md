# The Himalayas

**The Himalayas** is a [Unity](http://unity3d.com) project which shows 3D Mountains in Himalayan Range of Nepal. This application will also show current weather data of the Mountain location, 3 days of forecast and Wikipedia information about the Mountain. Also the mountains can be view in 3D mode, VR Mode and AR mode. *Mountain Locator* feature of this application will locate the mountain from user's device's persperctive.

## Dependencies

1. [SimpleJSON](http://wiki.unity3d.com/index.php/SimpleJSON)
2. [TouchKit](https://github.com/prime31/TouchKit)
3. [Gaia](http://www.procedural-worlds.com/gaia/)
4. [Vuforia SDK](https://vuforia.com)
5. [Google Cardboard SDK](https://developers.google.com/cardboard/unity/)
